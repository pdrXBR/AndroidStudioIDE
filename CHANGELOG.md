# Changelog

All notable changes to AndroidIDE will be documented in this file.

## [Unreleased]
- Git integration (commit, push, pull via Termux)
- On-device LLM via llama.cpp bindings
- Extension/plugin system (JSON-declared language servers)
- Code minimap rendering
- Vim keybinding mode

## [1.0.0] - 2024-01-01
### Added
- Initial release
- Sora Editor with TextMate syntax highlighting
- Python support via Termux + pylsp
- HuggingFace, Ollama, and offline AI assistants
- File tree navigator with CRUD operations
- Multi-tab editing with unsaved indicators
- Embedded terminal with interactive shell
- Command palette (VS Code-style)
- Settings: font, theme, AI, auto-save
- Global search in files
- GitHub Actions CI/CD pipeline
